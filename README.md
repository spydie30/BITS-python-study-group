# BITS Python Study Group

A repository for Python learning and practice exercises for the May Cohort Python study group.

## Files

- **assignemt.py** - Assignment operators and floor division
- **hello.py** - Hello world and basic examples
- **hellobits.py** -  hello world program

## Getting Started

To run any of the Python files, use:

```bash
python filename.py
```

## Requirements

- Python 3.x

## Usage

Each Python file contains different exercises and examples. Run them individually to see the output and learn from the code.
